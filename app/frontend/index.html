Uninterrupted Mode

        
Keep chatting even if a provider fails. When on, if your selected model errors (offline, rate-limited, overloaded, timed out, or no key), Chat Studio automatically retries the next provider in priority order — local MLX + your configured cloud providers.

        
        

          
          
          
If a provider sends no token within this time, fall back to the next one.
